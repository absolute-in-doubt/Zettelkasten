# Настройки font - CSS


### Size

```css
h1 {
	font-size: 1.15em;
}
```


---
### Weight (Boldness)

```css
h1 {
	font-weight: (0-1000); /* boldness */
}
```


---
### Font-family

```css
body {
  font-family: Arial, Helvetica, sans-serif; /* */
}
```


---
### Font-style

```css
h1 {
	font-style: italic;
}
```

| `normal`          | The default value. The text is shown in a regular face.                                                                                                                         |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `italic`          | Selects a font that is classified as _italic_ within the font-family. Italic typefaces often have unique, cursive-like character shapes designed by the font creator.           |
| `oblique`         | Selects a font that is classified as _oblique_ (a sloped version of the regular font). If a specific angle is not specified, a default angle (often around 14 degrees) is used. |
| `oblique <angle>` | (For variable fonts) Specifies an angle for the slant of the text, from `-90deg` to `90deg`.                                                                                    |
