# AOP - Spring

**Аспектно‑ориентированное программирование (АОП)** — это парадигма, которая дополняет объектно‑ориентированное программирование и позволяет **выделять и централизованно описывать сквозную функциональность** (например, логирование, безопасность, транзакции), которая обычно «размазывается» по множеству классов.
