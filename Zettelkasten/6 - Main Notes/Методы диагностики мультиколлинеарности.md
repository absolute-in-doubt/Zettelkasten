# 袦械褌芯写褘 写懈邪谐薪芯褋褌懈泻懈 屑褍谢褜褌懈泻芯谢谢懈薪械邪褉薪芯褋褌懈

### VIF (袦袠肖) 屑械褌芯写 懈薪褎谢褟褑懈芯薪薪褘褏 褎邪泻褌芯褉芯胁

(Variance Inflation Factor - 屑械褌芯写 "褉邪蟹写褍褌懈褟" 写懈褋锌械褉褋懈懈 褎邪泻褌芯褉芯胁)


1.  袠蟹 屑芯写械谢懈 锌械褉械斜芯褉芯屑 褋芯褋褌邪胁谢褟械屑 $m$ 褍褉邪胁薪械薪懈泄 (胁褋锌芯屑芯谐邪褌械谢褜薪褘褏 屑芯写械谢械泄)
#### $y = a_0 + ax_1 + \dots + a_mx_m + \varepsilon$

袛谢褟 $x_i$ 胁褋锌芯屑芯谐邪褌械谢褜薪芯械 褍褉邪胁薪械薪懈械 胁褘谐谢褟写懈褌 褌邪泻:
#### $x_i = b_0 +b_1x_1 + \dots + b_{i-1}x_{i-1} + b_{i+1}x_{i+1} + \dots + b_m + u_i$ 


2. 袙褘褔懈褋谢褟械屑 $R^2$ 写谢褟 泻邪卸写芯泄 懈蟹 胁褋锌芯屑芯谐邪褌械谢褜薪褘褏 屑芯写械谢械泄. 袠 褋褔懈褌邪械屑 懈薪褎谢褟褑懈芯薪薪褘械 褎邪泻褌芯褉褘:
#### $VIF_i = \frac{1}{1 - R^2}  \quad ,i = 1,m$

械褋谢懈 $5 <\text{min}(VIF) < 10$  => 褋褍褖械褋褌胁褍械褌 褍屑械褉械薪薪邪褟 屑褍谢褜褌懈泻芯谢谢懈薪械邪褉薪芯褋褌褜 (屑芯卸薪芯 褋屑懈褉懈褌褜褋褟 褋 薪械泄).

械褋谢懈 $\text{min}(VIF) > 10$ => 褋懈谢褜薪邪褟 屑褍谢褜褌懈泻芯谢谢懈薪械邪褉薪芯褋褌褜
