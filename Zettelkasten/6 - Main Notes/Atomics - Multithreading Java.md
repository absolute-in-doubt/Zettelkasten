# Atomics - Multithreading Java

`java.util.concurrent.atomic`

Атомики в Java работают как операции, которые гарантируют атомарность и видимость изменений в разделяемых переменных для многопоточных программ. 

Атомарные операции гарантируют, что операция будет выполнена полностью и никакой другой поток не сможет изменить значение переменной до завершения операции.

**~={red}Атомики не используют блокировки=~**. **Они обеспечивают атомарность через CAS — специальную атомарную инструкцию процессора, которая позволяет обновить значение, только если его никто не изменил между чтением и записью.**

> [!tip] **Что такое CAS?**
> (Compare And Swap): сравнил и, если версия больше, заменил.
