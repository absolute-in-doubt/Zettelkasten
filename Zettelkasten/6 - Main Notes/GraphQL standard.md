# GraphQL standard


- **Гибкая настройка получаемых данных**
	
- **Строгая типизация**
	

#### Минусы GrpahQL:

- Запросы могут сильно влиять на произволдительность серверс

- Все запросы отправляются в виде POST HTTP requests

- Все ответы имеют статус 200, даже в случае ошибок (error message - in the response body).
