# LocalTunnel

Запуск (требует node.js):
```bash
node ./bin/lt.js --port 3000
```

Открывает для внешнего доступа порт 3000.

Запускаетсяпри входе в саму папку с бинарниками (на WSL)

Мой путь:
```bash
neo_linux@DESKTOP-HT0FQJO:~/localtunnel/localtunnel/bin$ node lt.js --port 4242
```
