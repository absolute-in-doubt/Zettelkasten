# button - HTML


Тег `<button>` создаёт интерактивную **кнопку**, которая может использоваться для отправки формы, сброса формы или запуска произвольного JavaScript-кода.

### Основные типы кнопок (`type`)

- `type="submit"`  
  Кнопка отправки формы.

- `type="reset"`  
  Кнопка сброса значений полей формы к исходным.

- `type="button"`  
  Обычная кнопка, по умолчанию не выполняет действий (обычно в паре с JavaScript).

---

### Глобальные атрибуты

Тег `<button>` поддерживает стандартные глобальные атрибуты HTML, например:

- `id` — уникальный идентификатор элемента.  
- `class` — CSS-классы для стилизации.  
- `style` — встроенные CSS-стили.  
- `title` — всплывающая подсказка при наведении.  
- `hidden` — скрывает кнопку визуально и для взаимодействия.  
- `tabindex` — порядок фокуса при навигации с клавиатуры.

---
### Атрибут onclick + JS 

Переход через JS обработчик:
```html
<button type="button" onclick="doSomething()">Нажми меня</button>

<script>
  function doSomething() {
    alert('Сработало действие без формы');
  }
</script>
```

Переход через изменения JS свойства `location.href`:
```html
<button type="button" onclick="location.href='/some/page'">
  Перейти на страницу
</button>
```

> [!tip] **При работе с Thymeleaf используем его специальный атрибут**
> `th:onclick`

### Специфические атрибуты для `<button>`

- `type`  
  Тип кнопки: `submit`, `reset`, `button`.  

- `name`  
  Имя кнопки при отправке формы (пара ключ–значение).  

- `value`  
  Значение, отправляемое вместе с `name`, если форма отправляется этой кнопкой.  

- `disabled`  
  Делает кнопку неактивной (нельзя нажать, недоступна для фокуса).  

- `form`  
  Указывает `id` формы, с которой связана кнопка, даже если она находится вне тега `<form>`.  

- `formaction`  
  URL, на который отправляется форма при нажатии именно этой кнопки (переопределяет `action` формы).  

- `formenctype`  
  Кодировка данных формы при отправке:  
  - `application/x-www-form-urlencoded` (по умолчанию)  
  - `multipart/form-data`  
  - `text/plain`  

- `formmethod`  
  HTTP-метод отправки формы для этой кнопки: `get` или `post`.  

- `formnovalidate`  
  Отключает валидацию формы при отправке через эту кнопку.  

- `formtarget`  
  Цель для результата отправки формы:  
  - `_self` — текущее окно/вкладка (по умолчанию)  
  - `_blank` — новая вкладка/окно  
  - `_parent`, `_top` или имя фрейма  

- `autofocus`  
  Автоматически устанавливает фокус на кнопку при загрузке страницы (должен быть только один `autofocus` на странице).

---

### Примеры использования

```html
<!-- Обычная кнопка -->
<button type="button">Нажми меня</button>

<!-- Кнопка отправки формы -->
<button type="submit" name="action" value="save">Сохранить</button>

<!-- Отключённая кнопка -->
<button type="button" disabled>Недоступно</button>

<!-- Кнопка, отправляющая форму на другой URL -->
<button type="submit" formaction="/upload" formenctype="multipart/form-data">
    Загрузить файл
</button>
```
