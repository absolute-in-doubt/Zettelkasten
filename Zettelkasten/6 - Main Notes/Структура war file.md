# Структура war file


```
myapp/                 ← контекст приложения (/myapp)
├── index.jsp          ← корневые публичные ресурсы (HTML, JSP, статические файлы)
├── some-page.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
├── images/
│   └── logo.png
├── WEB-INF/           ← спецкаталог, не доступен напрямую по HTTP
│   ├── web.xml        ← дескриптор развёртывания (настройки сервлетов, фильтров, маппинги)
│   ├── classes/       ← .class-файлы и ресурсы приложения
│   │   └── com/example/... 
│   └── lib/           ← JAR-библиотеки, используемые только этим приложением
│       └── my-lib.jar
└── META-INF/          ← (опционально) метаданные приложения
    └── context.xml    ← (опционально) контекстные настройки
```
