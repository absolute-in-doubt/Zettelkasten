
Theory for the cards: [[Atomics - Multithreading Java]]

FILE TAGS: multithreading java_interview

Q: Расскажи про атомики в Java. Каким образом они предоставляют потокобезопасный доступ к данным?
A: Операции (методы) атомиков являются атомарными и предоставляют видимость изменений (visibility) для shared переменных потоков.
	
Атомарные операции гарантируют, что операция будет выполнена полностью и никакой другой поток не сможет изменить значение переменной до завершения операции.
	
**Атомики не используют блокировки. Они обеспечивают атомарность через CAS — специальную атомарную инструкцию процессора, которая позволяет обновить значение, только если его никто не изменил между чтением и записью.**
	
(Compare And Swap): сравнил и, если значение совпадает с изначальным (не было изменено другими потоками) - заменил на новое значение.
