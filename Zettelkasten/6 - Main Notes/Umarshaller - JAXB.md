# Umarshaller - JAXB

**Unmarshaller** – это компонент JAXB, который выполняет демаршалинг, преобразуя XMLдокумент в объект Java на основе аннотаций и настроек соответствующего класса.


### Методы:

- `Object` ~={green}.unmarshal(`File f`)
- `Object` .unmarshal(`InputStream is`)=~ - демаршаллит переданный XML, возвращает Object, который мы вручную type-castим
