# Вывод(подмена) текста и атрибутов - Thymeleaf

- **Вывод текста/атрибутов**:
    
    - `th:text` — заменить содержимое тега текстом.
        
    - `th:utext` — вставить HTML без экранирования.
        
    - `th:href`, `th:src`, `th:title` и др. — динамика для атрибутов.
