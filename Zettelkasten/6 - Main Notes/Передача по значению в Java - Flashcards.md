
Theory for the cards: [[Передача по значению в Java]]

FILE TAGS: java_important_moments

Q: Каким образом передаются переменные в методы, по значению или по ссылке?
A: В Java параметры всегда передаются только по значению, что определяется как «скопировать значение и передать копию». С примитивами это будет копия содержимого. Со ссылками - тоже копия содержимого, т.е. копия ссылки. При этом внутренние члены ссылочных типов через такую копию изменить возможно, а вот саму ссылку, указывающую на экземпляр - нет.
