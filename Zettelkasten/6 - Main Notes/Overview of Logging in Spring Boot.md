# Logging in Spring Boot


Использует пакет `spring-jcl`.


### `spring-boot-starter-logging`

~={cyan}(Содержится в `spring-boot-starter-web`)=~


В Spring Boot есть `LoggingSystem` - абстракция, которая конфигурирует реализацию логгирования, основываясь на содержимом `classpath`.

> [!note]
> Если в `classpath` есть несколько реализаций логгирования - **Logback** в приоритете
