# display - CSS

## Основные значения display

- `display: block;`  
  Элемент занимает всю доступную ширину, начинается с новой строки. Можно задавать ширину, высоту, отступы.

- `display: inline;`  
  Элемент идёт в одну строку с текстом и другими inline‑элементами. Нельзя полноценно задавать ширину/высоту.

- `display: inline-block;`  
  Как inline (в строку), но можно задавать ширину и высоту. Удобно для кнопок, иконок и т.п.

- `display: none;`  
  Элемент полностью убирается из потока и не отображается.

---

## Современные макеты

- `display: flex;`  
  Включает flex‑контейнер. Удобен для одномерных макетов (распределение по одной оси).

- `display: inline-flex;`  
  То же, что `flex`, но сам контейнер ведёт себя как inline‑элемент.

- `display: grid;`  
  Включает grid‑контейнер для двумерных сеток (строки и столбцы).

- `display: inline-grid;`  
  Grid‑контейнер, который ведёт себя как inline‑элемент.

---

## Табличные и прочие

- `display: table;`, `table-row;`, `table-cell;`  
  Эмуляция поведения таблиц для нестандартной разметки.

- `display: list-item;`  
  Элемент ведёт себя как элемент списка (`<li>`), может отображать маркеры.

- `display: contents;`  
  Сам элемент исчезает визуально и из потока, но его дети остаются и рендерятся так, будто они подняты на уровень выше.
