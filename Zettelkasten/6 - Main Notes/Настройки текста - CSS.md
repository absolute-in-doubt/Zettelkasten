# Настройки текста - CSS

### Text decoration

```css
.my-class {
	text-decoration-line: {none (default), underline, overline, line-through};
	
	
	text-decoration-color: black; /*Sets the color of the decoration line.*/
	
	text-decoration-style: {solid (default), double, dotted, dashed, wavy} /*Sets the style of the line*/
	
	text-decoration-thickness: {auto, from-font, <length>, <percentage>}
}
```


---
### Форматирование текста

```css
h1 {
	text-transform: {upper-case, lower-case, capitalize};
}
```


---
### Выравнивание 

```css
h2 {
	text-align: {center, left, right, justify};
}
```

`text-align: justify;` - Заполнить текстом всё выделенное пространство (ширину элемента)
