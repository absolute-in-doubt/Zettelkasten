# List implementations- Time Complexity

| Операция                         | ArrayList      | LinkedList                                 | ArrayDeque                          |
|----------------------------------|----------------|--------------------------------------------|-------------------------------------|
| Вставка в середину по индексу   | O(n)           | O(1) при известном узле / O(n) с поиском   | не поддерживается                   |
| Удаление из середины по индексу | O(n)           | O(1) при известном узле / O(n) с поиском   | не поддерживается                   |

| Удаление с конца                | O(1)           | O(1)                                       | O(1)                                |
