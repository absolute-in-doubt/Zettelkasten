# Таблицы - HTML

# HTML таблицы: основные теги и атрибуты

## 🔹 Основные теги внутри `<table>`
- `<tr>` — строка таблицы (table row).
- `<td>` — ячейка данных (table data).
- `<th>` — ячейка-заголовок (table header).
- `<thead>` — группа строк заголовка.
- `<tbody>` — основное тело таблицы.
- `<tfoot>` — нижняя часть таблицы (часто для итогов).
- `<caption>` — подпись таблицы.

---

## 🔹 Атрибуты тегов

### `<table>`
- `border` — толщина границы (устаревший, лучше через CSS).
- `cellspacing` — расстояние между ячейками (устаревший).
- `cellpadding` — отступ внутри ячеек (устаревший).
- `width` — ширина таблицы (лучше через CSS).

### `<tr>`
- `align` — выравнивание содержимого строки (устаревший).
- `bgcolor` — цвет фона строки (устаревший).

### `<td>` и `<th>`
- `colspan` — объединяет ячейку по горизонтали (в несколько колонок).
  - Пример: `<td colspan="2">Объединено</td>`  
    → ячейка занимает ширину двух колонок.
- `rowspan` — объединяет ячейку по вертикали (в несколько строк).
- `align` — горизонтальное выравнивание содержимого (устаревший).
- `valign` — вертикальное выравнивание содержимого (устаревший).
- `width`, `height` — размеры ячейки (лучше через CSS).

---

## 🔹 Пример использования `colspan`

```html
<table border="1">
  <tr>
    <th>Имя</th>
    <th colspan="2">Контакты</th>
  </tr>
  <tr>
    <td>Иван</td>
    <td>Email</td>
    <td>Телефон</td>
  </tr>
</table>
```
