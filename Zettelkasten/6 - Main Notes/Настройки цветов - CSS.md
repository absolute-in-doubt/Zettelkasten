# Настройки цветов - CSS


### Цвет текста

```css
h1 {
	color: black;
}

h2 {
	color: #F54927;  <- HEX color code
} 

h3 {
	color: rgb(0,0,0);
}
```


### Цвет фона

```css
h1 {
	background-color: rgb(1,1,1);
}
```


### Фоновое изображение

```css
.my-class {
	background: url("https://link-to-image");
	background-repeat: {repeat, no-repeat}; /* повторять ли image для заполнения всего пространства */
	
	background-size: y x;
}
```
