# Этапы выполнения Java кода - Java Core


Java‑код выполняется в два крупных этапа: сначала компиляция исходников в байткод, затем запуск байткода внутри виртуальной машины с предоставлением среды выполнения.

- Написание кода (.java) — разработчик + любые IDE/редакторы.

- Компиляция в байткод (.class) — компилятор `javac`.

- Загрузка классов — подсистема загрузчиков классов (Class Loader, часть JVM). 
  При запуске загружается главный класс и по мере надобности остальные классы через механизм загрузчиков классов (class loaders)

- Проверка байткода — верификатор байткода (Bytecode Verifier, внутри JVM).

- Интерпретация и JIT-компиляция — интерпретатор и JIT-компилятор (в составе JVM).

- Выполнение и управление памятью — JVM + сборщик мусора (Garbage Collector).

- Предоставление стандартных API — стандартная библиотека Java (rt / модули JDK).
