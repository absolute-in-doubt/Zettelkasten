# Признаки мультиколлинеарности


1. Оценки коэффициентов по МНК не являются [[Оценка качества точечной оценки параметра СВ|несмещенными]]

2. Дисперсия оценок коэффициентов по МНК возрастает (след из п.1)

3. Уменьшение [[t-статистика (t-критерий Стьюдента)|t-статистик]] значимости оценок параметров по МНК => ложные выводы о незначимости коэффициентов

4. $R^2$ - остаётся высоким (~={orange}не показатель мультиколлинеарности=~)

5. Оценки очень чувствительны к изменению объёма выборки

6. Оценки параметров при НЕколлинеарных экзогенных переменных становятся незначительными
