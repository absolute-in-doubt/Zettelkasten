FILE TAGS: databases postgres

Q: Какие ты знаешь типы индексов в Postgres?
A: Самые популярные: B-Tree и Hash индекс
1. B-Tree индекс 
	- **Как работает:** Хранит данные в виде сбалансированного дерева, позволяя быстро находить записи по ключу за O(log n). 
	  **Используется по умолчанию в Postgres.** 
	- **Когда применять:** Подходит для большинства запросов с `WHERE`, `ORDER BY`, `GROUP BY`, `JOIN`, особенно, если условия используют операторы =, <, >, `BETWEEN`.
2. Hash индекс
	- **Как работает:** Использует хеш-таблицу для юыстрого поиска **точечных совпадений** (=) за O(1)
	- **Когда применять:** Идеален для поиска по точному совпадению (`WHERE column = value`), но не эффективен для поиска по диапазонам (`BETWEEN`, >, <) или сортировки
