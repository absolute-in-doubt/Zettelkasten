# Когда использовать ВР

Временные ряды (включая авторегрессионные модели вроде AR, ARIMA, VAR) применяют при данных с временной зависимостью (автокорреляцией), где прошлое влияет на будущее. Регрессионные модели (OLS, GLM) подходят для независимых наблюдений без структуры по времени или с доминирующими внешними предикторами.​

## Ключевые случаи для временных рядов

- Высокая автокорреляция (ACF/PACF значимы на ACF >0.1 для лагов).​

- Динамика: тренд, сезонность, циклы (денежные потоки, акции, продажи).​

- Прогнозирование последовательностей (финансы, погода, спрос).[​

- Многомерные взаимодействия (VAR для потоков+ставок).[​


## Ключевые случаи для регрессии

- Независимые наблюдения (кросс-секшн: зарплаты по регионам).​

- Каузальные связи с фиксированными X (возраст влияет на доход)​.

- Отсутствие лагов (Durbin-Watson ~2).[​


## Сравнение применения

| Критерий       | Временные ряды (ARIMA/VAR)        | Регрессия (OLS)                                                                         |
| -------------- | --------------------------------- | --------------------------------------------------------------------------------------- |
| Автокорреляция | Высокая (лаги значимы) ​          | Низкая/отсутствует​                                                                     |
| Тип данных     | Последовательные по времени ​     | Независимые (cross-section) ​                                                           |
| Цель           | Прогноз динамики​                 | Объяснение связей​                                                                      |
| Диагностика    | ADF (стационарность), Ljung-Box ​ | Durbin-Watson ​                                                                         |
| Пример         | Денежные потоки AR(3)             | Зависимость:                                        Потоки ~ инфляция (без лагов) <br>​ |

## Практический выбор

Проверьте автокорреляцию (ACF/PACF), стационарность (ADF-тест) и остатки. Если Durbin-Watson ≠2 или Breusch-Godfrey p<0.05 — переходите к временным рядам. Гибрид: ARIMAX с регрессорами.
