Apache Velocity - templaate enginge, который облегчает процесс генерации динамического контента. Позволяет отделить Java код от конечного представления (presentation layer).

Имеет лёгкий синтаксис, позволяющий non-developers модифицировать представление (presentation layer) без залезания глубоко в код.
