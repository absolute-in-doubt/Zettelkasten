# Как хранятся сообщения - Kafka

Сообщения в каждом топике хранятся по маленьким сегментам. Каждый сегмент сохраняется в свой файл.

![[Pasted image 20260207094706.png]]

Эти файлы надо периодически удалять. Для этого существуют настройки TTL.

`retention.bytes` = {-1, -2} - что делает хз
`retention.ms` - задаёт кол-во милисекунд, котрое хранится каждый сегмент, после чего он будет удалён.

![[Pasted image 20260207095000.png]]

- В `.log` сохраняется - какое сообщение куда сохранено.
- В `.index` - сохранение offset и size.
- `.timeindex` - когда был записан тот или иной offset в файл. (timestamp - offset)
