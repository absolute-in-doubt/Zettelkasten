# Алгоритм динамического свзяывания, vtable класса - Java Core


Каждый класс хранит таблицу методов: в ней по фиксированным индексам лежат указатели (адреса) на реализации виртуальных методов класса

Объект в куче хранит скрытую ссылку на свой `Class` (метаданные), откуда косвенно можно добраться до его таблицы виртуальных методов.


## Формирование vtable при наследовании

- Для базового класса строится таблица, содержащая все его виртуальные методы (в Java практически все нестатические методы виртуальные, кроме `private`, `static`, `final`).​
    
- Для подкласса создаётся своя таблица на основе таблицы родителя:
    
    - методы, которые не переопределены, просто копируют записи родителя;
        
    - переопределённые методы _заменяют_ соответствующие записи: в той же позиции указывают на новую реализацию.
