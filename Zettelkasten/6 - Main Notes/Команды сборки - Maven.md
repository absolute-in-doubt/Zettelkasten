# Команды сборки - Maven

- `mvn clean` - очищает target
- `mvn compile` - компиляция
- `mvn test` - выполенение тестов
- `mvn package` - просто сборка jar
- `mvn install` - сборка + загрузка этого jar в локальный репозиторий (чтобы другие проекты могли использовать его как зависимость)
