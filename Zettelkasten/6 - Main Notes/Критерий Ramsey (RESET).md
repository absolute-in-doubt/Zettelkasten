# Критерий Ramsey (RESET)

Используется для проверки на пропущенную переменную.

Это обязательная часть анализа модели вместе с проверкой выполнения предпосылок МНК.

---
### Гипотезы F-статистики, проверяемые критерием Ramsey:

### $H_0: \tilde{y^2}, \tilde{y^3}, \tilde{y^4} ... \tilde{y^k} = 0$

### $H_1: \exists \tilde{y^n} \ne 0$


---
### Алгоритм действий

1. МНК для начальной модели:
#### $\tilde{y_t} = \tilde{a_0} + \tilde{a_1}x_{1t} + \tilde{a_2}x_{2t}$

2. анализируем вид зависимости остатков (Для проверки выполнения ключевых предпосылок МНК)

3. с учётом шага 2 вычисляем величины $\tilde{y^2}, \tilde{y^3}, \tilde{y^4}$:

`+` подставляем их в модель. Если коэффициенты при этих параметрах будут незначимы -> в этой модели нет пропущенных переменных.

#### $y_t = a_0 + a_1x_{1t} + a_2x_{2t} + a_3\tilde{y_t^2} + a_4\tilde{y_t^3} + a_5\tilde{y_t^4} + \varepsilon$

4. Считаем F-статистику критерия Ramsey:

### $F = \frac{(RSS_1 - RSS_2)/k}{RSS_2/(T-m-1)}$

где 
- $RSS_1, RSS_2$ - сумма квадратов отклонений начальной модели и модели с подставленными $\tilde{y^2}, \tilde{y^3}, \tilde{y^4}$ соответственно.
- T - количество измерений (вариант)
- $m$ - количество регрессоров во второй модели (с подставленными $\tilde{y^2}, \tilde{y^3}, \tilde{y^4}$)
- $k$ - количество подставленных регрессоров

5. Сравниваем качество полученной модели с помощью F-статистики:

#### $F_\text{набл} > F_\text{крит}(\alpha, k, T - m -1)$   ->  отвергаем $H_0$ -> значит, пропущена переменная
