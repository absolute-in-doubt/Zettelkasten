# Модификатор метода final - Java

Используется на методах, которые не должны быть переопределены в подклассах.

> [!note]
> Final методы resolve-ятся статически, т.е. на этапе компиляции. (Т.к. нет смысла ждать, какой тип его вызвает, ведь вызвать его может только один класс, а не его подклассы)
