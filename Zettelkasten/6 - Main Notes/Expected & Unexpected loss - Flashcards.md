
Theory for the cards: [[Expected & Unexpected loss]]

FILE TAGS: FRM

Q: Distinguish between expected loss and unexpected loss and provide examples of each.
A: **Expected loss** - оценка потерипри нормальном положении рынка. Может быть численно оценена. 
	
Формула Expected losses (EL):
	
$EL = EAD \times PD \times LGD$
	
где
- PD (%) - probability of default (субъекта в который инвестировали)
	
- EAD ($) - exposure at default - сумма долга (сколько нам должны) на момент дефолта
	
- LGD (%) - magnitude of the loss given default - доля потерь при дефолте
	Например, если клиент не вернул $100 ,000, но банк смог вернуть $40, 000 через залог, то LGD = 60%.
	
> [!note]
> Когда EL может быть достаточно точно рассчитан, он может бытть включён в себестоимость.
> 
	
Пример:
	
Банки повышают проценты для займополучателей с более высоким уровнем риска (и уменьшают максимальный срок займа).
	
---
**Unexpected loss** - потери свыше Expected loss.
	
Пример:
	
При выдаче займов с залогом в виде имущества при нормальном рынке дефолты займополучателей будут на определённом уровне, учитывая который мы сможем посчитать EL. Но при падении цен на недвижимость многие получатели займов обанкротятся. Таким образом, этот **корреляционный риск** (два риск фактора происходящих одновременно) могут вывести потенциальные потери выше ожидаемого уровня -> Unexpected loss.
