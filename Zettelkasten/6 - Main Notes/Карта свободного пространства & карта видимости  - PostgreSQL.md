### Карта видимости

Индекс не знает изначально, видна ли запись, эта инфа лежит в Heap вместе с данными (`txmin`, `txmax`).

![[Pasted image 20260214173200.png]]


Каждая ячейка - страница. Зеленая ячейка - значит на странице ВСЕ данные актуальны - данные видны абсолютно всем транзакциям.

Биты значений (red, green) устанавливаются лениво - при [[Очистка { AUTOVACUUM } + очистка индексов - PostgreSQL|VACUUM]] - VACUUM пропылесосил -> поставил зелёный свет. Обновляются при INSERT, UPDATE, DELETE. Может отставать от реальности, но никогда не врёт.
